Kopyala - Yapıştır - Kes Komutları :

CTRL + C ile kopyalama, CTRL + V ile yapıştırma, CTRL + X ile kesme işlemleri yapılabilir. Bu işlemler eğer o satırda herhangi bir alan seçilmeden yapılırsa tüm satıra uygulanır.

Geri Alma - Ileri Sarma :

CTRL + Z tuşlarıyla bir önceki işleme geri dönülebilir, SHIFT + CTRL + Z ile geri alma işlemi ileri sarılır, yani geri alma işlemi geri alınır.

Kelime kelime Ilerlemek ve Seçim Yapmak :

CTRL + SOL/SAĞ OK ile satırda kelime kelime ilerleme yapılabilir.

SHIFT + CTRL + SOL/SAĞ OK ile seçim yapılarak kelime kelime ilerlenir. Birden fazla basılarak kod bloğunun seçimi arttırılır

Tüm Satırın Fare ile Seçilmesi :

Sol Click 3 kere hızlıca yapılırsa o satır seçilir.

Satır Numarasına Gidilmesi :

CTRL + g ile satır numarasına gidilir.

Satırın Taşınması :

ALT + YUKARI/ASAGI OKLAR : imlecin oldugu satır yukarı/aşağı taşınabilir.

Birden Fazla Imleç Oluşturmak:

ALT + SOL Click ile birden fazla imleç oluşturulabilir.

Seçilen Alanın veya Satırın Yorum Alanına Dönüştürülmesi :

CTRL + K + C ile seçilen alanı yorum satırlarına dönüştürülür. Bunu satır satır yapar.

CTRL + ö ise sadece tek satırı yorum satırı yapar. Ayrıca yorum satırı olan alanın yorum satırından çıkarılması için de kullanılabilir.

Siz yorum bloğu oluşturmak istiyorsanız SHIFT + ALT + A kısayolunu kullanabilirsiniz.

Satırın Alt veya Üst Satıra Kopyalanması :

SHIFT + ALT + YUKARI/ASAGI OKLAR ile imlecin olduğu satır kopyalanır.

En Son İşlem Yapılan Satıra Gidilmesi

ALT + SOL/SAĞ OKLAR ile en son işlem yaptığınız satırlar arasında gezinebilirsiniz.

Dosya Arama :

CTRL + P tuşlarıyla hızlıca dosya arayıp içine girebilirsiniz.

Dosya İçinde Arama :

CTRL + F dosya içinde arama yapar.

Sol Kısmı Kapatıp Açma :

CTRL + B tuşları editörün sol kısmındaki alanı kapatıp açar.

Açık Tablar Arasında Gezinme :

CTRL + TAB bir sonraki sekmeye geçer, CTRL + SHIFT + TAB bir önceki sekmeye geçer.

Önerilerin Sarı Ampülün Çıkması :

CTRL + . ile ilgili yerdeki önerilerin çıkması sağlanır.( Bazen sarı ampülün çıktığı yerlerde )

Süslü Parantez Sonuna Gidilmesi :

CTRL + SHIFT + ç ile süslü parantezin bittiği yere gidilebilir.

Terminal Penceresini Açmak :

CTRL + "
